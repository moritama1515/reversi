reversi
====
リバーシのプログラムです．
今のところCとjavascriptのプログラムがあります．

## Description
Cのプログラムは基本機能は実装してあります．
javascriptのプログラムはCをトレースしてるのでオブジェクト指向とか全然考えてないです．(開発中)


## Usage
XとYに数字を入力してください．
禁じ手や盤外，数字以外の場合はもう一度入力を求められます．
